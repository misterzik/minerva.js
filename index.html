Product Overview

Minerva.js is a lightweight jQuery Library that will help you, wrap around your images, into a slideshow or sort of like carousel order.

Installations

2. Lets run Slider wrapper
3. Lets Populate/Add as many images as, We Please
4. Lets run the Dots wrapper/efx
5. Don't forget to Add as Many Dots, As needed.
6. Happy Coding
